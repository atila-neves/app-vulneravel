# Aplicação de Teste Vulnerável
Projeto criado para testar o conector BNVD com vulnerabilidades reais.

Contém:
- Flask desatualizado (vulnerável)
- Requests vulnerável
- PyYAML vulnerável
- Execução de comandos via parâmetro
- YAML insecure load

Para rodar:
```
pip install -r requirements.txt
python app.py
```
